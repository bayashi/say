=head1 INSTALLATION

say installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan say

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install


=head1 DOCUMENTATION

say documentation is available as in POD. So you can do:

    % perldoc say
